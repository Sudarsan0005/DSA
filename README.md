# DSA

# 🚀 150-Day DSA Roadmap (Beginner → Intermediate → Strong Interview Prep)

This version is built for **1 to 1.5 hours per day** and includes **all problems from your source list**.



---

# Daily Rule

| Day Type | Target |
|---|---|
| Easy-topic day | 2–3 problems |
| Medium-topic day | 1–2 problems |
| Hard-topic day | 1 problem |
| Revision day | Re-solve old problems, no new pressure |

## Daily split
- 15 min: revise notes / template
- 35–45 min: solve 1 problem
- 15–25 min: solve 2nd problem or review editorial
- 10 min: write mistake notes
